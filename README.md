# Image Resizer

Resize and compress images instantly in your browser. No uploads required.

## Features
- Batch image resizing
- Custom dimensions
- Format conversion (PNG, JPG, WebP)
- Quality control
- Download resized images
- Mobile-friendly

## Tech Stack
- HTML/CSS/JavaScript
- Canvas API
- Cloudflare Pages

## Live Demo
https://image-resizer.pages.dev

## Run Locally
Open index.html in your browser.
